# AGENTS.md

## Project

This is the official academic laboratory website for Prof. Shih-Chen Shi's Advanced Material and Processing Lab (AMPL), Department of Mechanical Engineering, National Cheng Kung University.

The site must look professional, academic, modern, refined, and trustworthy.

## Core Research Identity

Design and content should reflect:

- Green synthesis
- Circular economy
- Nanomaterials
- Biopolymer composites
- Surface engineering
- Tribology
- Biomedical materials
- Environmental materials
- Sensing materials
- Simulation-assisted material and process optimization

## Design Standard

Do not produce a generic template-looking website.

The UI must have:

- strong visual hierarchy
- clean academic layout
- consistent spacing
- polished desktop and mobile views
- subtle animations
- readable typography
- restrained colors
- refined research-oriented visuals
- accessible interactions

Avoid:

- random gradients
- excessive animation
- cartoon icons
- low contrast
- crowded sections
- inconsistent card sizes
- fake content
- placeholder-heavy final presentation
- startup-style exaggerated hero sections
- generic AI-looking design

## Preferred Visual Direction

Use a premium academic materials-science style:

- white / off-white background
- deep navy / muted blue primary palette
- graphite text
- restrained green/gold accents
- abstract microstructure / molecular lattice / material layer visuals
- subtle cards
- soft borders
- generous spacing
- precise alignment

## Suggested Tech Stack

Use the existing project stack if present.

For a new project, prefer:

- Astro or Next.js
- TypeScript
- Tailwind CSS
- component-based structure
- MD/MDX or data files for content
- CSS animations or a lightweight motion library

Do not introduce unnecessary backend complexity.

## Content Rules

Use verified public information only.

Do not invent:

- student names
- member photos
- equipment names
- unpublished awards
- publications
- grants
- lab facilities
- contact details

Use explicit placeholders:

- [待提供：成員名單]
- [待提供：設備照片]
- [待確認：代表論文]
- [待確認：實驗室 Logo]

## Homepage Requirements

The homepage must include:

1. Hero section:
   - 尖端材料及製程實驗室
   - Advanced Material and Processing Lab
   - AMPL
   - Green Synthesis · Circular Economy · Nanomaterials · Surface Engineering

2. Short research identity statement.

3. Animated abstract material visual:
   - microstructure
   - molecular lattice
   - layered material
   - particle network

4. Research area cards.

5. Materials-to-applications pipeline:
   - Waste / Biomass
   - Nanomaterials
   - Processing & Coating
   - Characterization / Simulation
   - Applications

6. Selected highlights / publications placeholder.

7. Join Us CTA.

## Animation Requirements

Animations must be subtle, performant, and accessible.

Include:

- sticky header transition
- hero text entrance
- animated material lattice / particle network
- scroll reveal
- card hover lift
- CTA hover motion
- mobile menu transition
- publication item hover

Respect `prefers-reduced-motion`.

Do not use excessive or distracting animation.

## Responsive QA

Before finalizing, inspect:

- 1440px desktop
- 1280px desktop
- 1024px tablet
- 768px tablet
- 430px mobile
- 390px mobile
- 360px mobile

Fix:

- horizontal scroll
- broken nav
- cramped hero
- inconsistent card heights
- low contrast
- mobile overflow
- bad wrapping
- unreadable publication list
- footer layout issues

## Accessibility

- Semantic HTML
- Valid heading hierarchy
- Visible focus states
- Keyboard accessible navigation
- Alt text for images
- ARIA labels for icon-only controls
- Good color contrast
- Reduced motion support

## Done When

The task is complete only when:

- the app builds successfully
- all required pages exist
- desktop and mobile layouts are visually acceptable
- navigation works
- animations are subtle and accessible
- no fake content is introduced
- placeholders are clearly marked
- visual QA screenshots or notes are provided
- run instructions are documented
