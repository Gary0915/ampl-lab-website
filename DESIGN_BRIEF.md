# AMPL Website Design Brief

## Brand Sentence

尖端材料及製程實驗室以綠色合成、循環經濟與先進材料製程為核心，發展高性能且具永續價值的奈米材料、複合材料、功能性表面與感測/生醫應用。

## Visual Keywords

- academic precision
- advanced materials
- sustainable process
- microstructure
- molecular lattice
- surface layers
- quiet confidence
- refined engineering

## Do

- Use structured whitespace
- Use subtle scientific abstract visuals
- Use deep navy and off-white
- Use restrained green and gold accents
- Keep animations slow and elegant
- Design placeholders intentionally
- Make publications highly readable
- Prioritize desktop and mobile polish

## Do Not

- Use random lab stock photos
- Invent real-world content
- Use neon sci-fi effects
- Use excessive gradients
- Use cartoon icons
- Overload every section with animation
- Leave default template styling
- Allow mobile overflow

## Suggested Hero Visual

A right-side animated SVG lattice:

- 18–32 nodes
- thin connecting lines
- low opacity
- 2–3 accent nodes
- slow vertical floating
- subtle radial glow
- no heavy canvas needed

## Suggested Section Visuals

Research cards:

- top accent line
- small abstract icon
- tag pills
- hover lift

Pipeline:

- horizontal desktop timeline
- vertical mobile timeline
- animated connector line
- short node descriptions

Publications:

- year column
- title and venue
- left border hover
- tags for topic/category

Facilities:

- image cards when photos exist
- polished placeholders when missing
