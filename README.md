# AMPL Academic Laboratory Website

## Run locally

```powershell
npm install
npm run dev
```

Open `http://localhost:4321`.

## Verify

```powershell
npm run build
npm test
```

Traditional Chinese pages are the default (`/`); English equivalents are under `/en/`.

## Content handoff needed

- Verified current member list and research topics
- Verified publication bibliography and highlighted works
- Facility inventory and photographs
- Laboratory logo, office location, map, and verified address
- Laboratory news and announcements
