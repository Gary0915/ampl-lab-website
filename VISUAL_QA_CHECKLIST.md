# AMPL Website Visual QA Checklist

Use this checklist before reporting completion.

## Build

- [ ] dependencies installed
- [ ] project runs locally
- [ ] production build passes
- [ ] no console errors
- [ ] no obvious hydration errors

## Pages

- [ ] Home
- [ ] About
- [ ] Research
- [ ] Members
- [ ] Publications
- [ ] Facilities
- [ ] News
- [ ] Join Us
- [ ] Contact

## Desktop QA

Check at 1440px and 1280px:

- [ ] hero spacing looks premium
- [ ] nav alignment is correct
- [ ] hero visual does not overpower text
- [ ] research cards align and have consistent heights
- [ ] pipeline layout is readable
- [ ] publication items are readable
- [ ] footer is complete and balanced
- [ ] no excessive blank areas
- [ ] no overcrowded areas

## Tablet QA

Check at 1024px and 768px:

- [ ] nav adapts correctly
- [ ] card grid wraps cleanly
- [ ] hero remains readable
- [ ] no horizontal overflow
- [ ] section padding is not excessive

## Mobile QA

Check at 430px, 390px, 360px:

- [ ] no horizontal scroll
- [ ] mobile menu opens and closes
- [ ] hero height is reasonable
- [ ] CTA buttons wrap cleanly
- [ ] research cards stack cleanly
- [ ] pipeline becomes vertical
- [ ] publication list remains readable
- [ ] footer does not look cramped

## Animation QA

- [ ] hero animation is subtle
- [ ] scroll reveal is not excessive
- [ ] hover states are smooth
- [ ] mobile menu animation is smooth
- [ ] reduced motion is respected
- [ ] animation does not obscure text

## Content QA

- [ ] no fake students
- [ ] no fake equipment
- [ ] no fake publications
- [ ] no fake awards
- [ ] placeholders are clearly marked
- [ ] lab/professor facts are separated from placeholders
